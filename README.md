# Enaria MX

Plataforma de trading impulsada por inteligencia artificial diseñada para el mercado mexicano. Enaria MX integra análisis predictivo, ejecución automatizada, gestión de riesgos y visualización 3D para ofrecer una experiencia moderna y confiable a traders e inversionistas.

## Qué es Enaria MX

Enaria MX es una suite de herramientas de IA que analiza patrones de mercado, predice tendencias y ejecuta operaciones con baja latencia. La aplicación presenta un interfaz cuidada con efectos y gráficos 3D, además de componentes interactivos que facilitan la exploración de datos reales (incluyendo integración con cuentas auditadas como MyFXBook).

## Características Clave

- Análisis predictivo con modelos de IA
- Trading automatizado 24/7 basado en estrategias configurables
- Gestión de riesgos con stop‑loss dinámico y métricas de drawdown
- Visualización 3D avanzada (velas, líneas de tendencia, partículas)
- Integración con fuentes de datos reales (MyFXBook)
- Interfaz responsiva con animaciones y efectos de entrada

## Arquitectura Técnica

- Frontend: `React 18` + `TypeScript`
- Build y dev server: `Vite 6`
- Estilos: `TailwindCSS 3`
- Gráficos 3D: `three`, `@react-three/fiber`, `@react-three/drei`
- Estado: `zustand`
- Linter: `eslint` con reglas para React/TS

## Scripts

- `npm run dev` — Inicia el servidor de desarrollo
- `npm run build` — Compila para producción
- `npm run preview` — Sirve la build producida
- `npm run check` — Verificación de tipos (TypeScript)
- `npm run lint` — Revisión de estilo y buenas prácticas

## Puesta en Marcha

1. Instala dependencias: `npm install`
2. Desarrollo: `npm run dev` y abre `http://localhost:5173/`
3. Producción: `npm run build` y luego `npm run preview`
